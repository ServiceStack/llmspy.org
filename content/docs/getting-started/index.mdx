## Using pip

The easiest way to install llms.py is using pip:

<ShellCommand>pip install llms-py</ShellCommand>

## Using Docker

### Pre-built Docker Images

Pull and run the latest image:

<ShellCommand>docker pull ghcr.io/servicestack/llms:latest</ShellCommand>

<ShellCommand>docker run -p 8000:8000 -e GROQ_API_KEY=$GROQ_API_KEY ghcr.io/servicestack/llms:latest</ShellCommand>

### Using docker compose (Recommended)

1. Download the docker-compose.yml file:

<ShellCommand>curl -O https://raw.githubusercontent.com/ServiceStack/llms/refs/heads/main/docker-compose.yml</ShellCommand>

2. Create a `.env` file with your API keys:

```bash
OPENROUTER_API_KEY=sk-or-...
GROQ_API_KEY=gsk_...
GOOGLE_FREE_API_KEY=AIza...
```

3. Start the service:

<ShellCommand>docker compose up -d</ShellCommand>

### Build from Source

If you've cloned the repository:

<ShellCommand>git clone https://github.com/ServiceStack/llms</ShellCommand>

<ShellCommand>cd llms</ShellCommand>

<ShellCommand>docker compose -f docker-compose.local.yml up -d --build</ShellCommand>

## Setup

[Setup environment variables](/docs/getting-started/quick-start#1-set-api-keys) for the providers you want to use.

## Updating

To update to the latest version:

<ShellCommand>pip install llms-py --upgrade</ShellCommand>

For Docker:

<ShellCommand>docker pull ghcr.io/servicestack/llms:latest</ShellCommand>
